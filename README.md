# 🎬 Movie Explorer Web App

## 📌 Project Description

The Movie Explorer Web App is a web application that allows users to search and explore movies using a public API. It dynamically fetches and displays movie data and provides interactive features like search, filtering, and sorting.

## 🎯 Purpose

The purpose of this project is to demonstrate the use of JavaScript, API integration using fetch, and UI design using CSS.

## 🔌 API Used

OMDb API (https://www.omdbapi.com/)

## ✨ Planned Features

* Search movies by name
* Filter movies by year or type
* Sort movies alphabetically or by release year
* Responsive UI design
* Loading indicator while fetching data

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)
* GitHub

## 🚀 How to Run

1. Clone the repository
2. Open index.html in browser
